Projects

Data Analytics

Analysis of Interval-Valued Data and prediction of electricity consumption in France.

Abstract
Each bar of this graph is an interval of the monthly electricity consumption. The lower and the upper bound represent respectively the minimum and the maximum value. This project first aimed to analyze different regression methods based on Interval Data and picked the one that has the lower prediction error. Second, each interval is transformed into a signle value data by taking the mid-point of each interval. Since there was a rigid seasonality, I modeled the time series using Fourier series. Then, the confidence interval at 5 and 1% has been calculated and compared with the true interval data. In conclusion, the prediction error was higher when performing the forecasting on the classical data and then find the confidence interval than to work directly with the interval data. In presence of interval data, we should analyze it without any prior transformation.
Keywords:Density function, Maximum Likelihood, Regression, Fourier series.

Data Warehousing

Building A Relational Online Analytical Processing System using YouTube And Ted Talks Datasets

Abstract
This paper aimed to integrate two system of records into a Data Warehouse following Kimball methodology. We have defined six Key Performance Indicators to support data sources that are represented by YouTube videos and TED talks from 2006 to 2018. SQL Server has been chosen as a target database to support data marts, and it is hosted by Microsoft Azure. Before running the ETL, some dimensions have been cleaned using Tableau Prep. The ETL process has been implemented on eleven of the thirty-four subsystems as defined by Kimball. After loading the data into the Data Warehouse, Tableau has been connected to Microsoft SQL Server, and we came up with meaningful insights from the data using some visualization based on comparative analysis, trending analysis, or correlation analysis. Keywords: Data Integration, ETL, Cloud Computing, Relational Databases, Business Intelligence, Key Performance Indicators
